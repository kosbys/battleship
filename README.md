 Vite Template </h1>
My template to get started quickly on projects

# :blue_book: Get started

```bash
git clone git@github.com:kosbys/vite-template.git
cd vite-template
npm install
npm run dev
```

##  :wrench: Tools

- <a href="https://vitejs.dev/"> Vite 3.0</a>
- <a href="https://tailwindcss.com/">TailwindCSS</a>
- TypeScript
- ESLint + Prettier
